- HTML5
- CSS3
- JavaScript
- PWA (Service Worker + Manifest)

## 📱 Funcionalidades
- Cardápio digital responsivo
- Instalação como app (PWA)
- Funcionamento offline
- Interface otimizada para mobile
- Landing page para apresentação
